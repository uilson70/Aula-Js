🚀 Principais Ferramentas para Iniciantes em JavaScript

Olá! Aqui está um guia prático e interativo para as ferramentas essenciais para quem está começando com JS. Cada seção tem exemplos que você pode testar agora mesmo. Salve este arquivo como HTML e abra no navegador!

1. Console do Navegador (F12) - A primeira ferramenta!

Aperte F12 no navegador (Chrome/Firefox) > aba Console. É onde você testa JS rapidinho.

Teste agora: Abra F12, cole isso no console e aperte Enter:

console.log("Olá, JS!"); let nome = "Usuário"; console.log(`Bem-vindo, ${nome}!`);

2. Editor VSCode - Seu workspace principal

Você já usa! Instale extensões: "Live Server", "ES7+ React/Redux", "Prettier".

Criar arquivo .js, aperte Ctrl+F5 com Live Server para rodar.

3. Node.js e npm - Rodar JS no terminal

Baixe em nodejs.org. Verifique: node -v e npm -v.

Crie um arquivo test.js e rode node test.js no terminal.

4. Editores Online (CodePen, Replit, JSFiddle)

Sem instalar nada! Perfeito para protótipos.

  • CodePen - HTML/CSS/JS
  • Replit - Projetos completos
  • JSFiddle - Rápido

5. Gerenciador de Pacotes: npm/yarn

Para bibliotecas como Express ou React mais tarde.

npm init -y
npm install lodash
